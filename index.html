Website Modifications

In this project, I customized the template provided in the following ways:

  • Modified the CSS to align with a One Piece theme, including custom colors and font styles.
  • Replaced template images with character-specific images for each Straw Hat Pirate.
  • Fixed navigation issues and ensured proper alignment across devices.
  • Removed unnecessary template elements (e.g., unrelated background images) and added a logo to the header.
  • Implemented a JavaScript feature to display random quotes on the home page.
  • Resized and styled character images for a cleaner look.
